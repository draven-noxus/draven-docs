# Hbase

```shell
# 官网
http://hbase.apache.org/

# 下载地址
http://hbase.apache.org/downloads.html

# 解压目录
tar -xzvf hbase-2.2.5-bin.tar.gz
mv ./hbase-2.2.5 /opt/hbase-2.2.5

# 配置hbase-site.xml

# 启动
./start-hbase.sh

# 查看进程
Jps

# shell命令
hbase shell

# 访问页面
http:ip:16010

# 初始化语句
./hbase shell hbase-create.hbase

# 参考链接
```
