2.JDK的上传和安装以及环境变量配置
 	上传
		查看系统JDK：java Cversion
	删除
		rpm -e --nodeps java-1.7.0-openjdk-1.7.0.45-2.4.3.3.el6.x86_64
	位置
		/usr/local/(**)
		eg:/usr/local/java
	解压

		yum install glibc.i686  安装jdk的依赖包（可选） yum表示在线安装
		解压：tar -xvf jdk-7u71-linux-i586.tar.gz -C /usr/local/java
	配置
		打开文件进行环境配置
		vi /etc/profile
		输入
		#set java environment
		JAVA_HOME=/usr/local/java/（*）
		JAVA_HOME=/usr/local/java/jdk1.7.0_71
		CLASSPATH=.:$JAVA_HOME/lib.tools.jar
		PATH=$JAVA_HOME/bin:$PATH
		export JAVA_HOME CLASSPATH PATH
	保存退出
	使更改的配置立即生效（重新加载）
		source /etc/profile 
	
	

3.Mysql的上传和安装..
	上传	
	
	检查是否存在
		 rpm -qa | grep mysql  
	卸载
		rpm -e --nodeps  mysql-libs-5.1.71-1.el6.x86_64
	解压	
		tar -xvf MySQL-5.6.22-1.el6.i686.rpm-bundle.tar
	依赖	
		yum -y install libaio.so.1 libgcc_s.so.1 libstdc++.so.6 
	是否需要升级
		yum  update libstdc++-4.4.7-4.el6.x86_64
	安装服务器
		rpm -ivh MySQL-server-5.6.22-1.el6.i686.rpm
		内含密码
	安装客户端
		rpm -ivh MySQL-client-server-5.6.22-1.el6.i686.rpm
		两个
	查看启动状态
		service mysql status
	启动
		service mysql start
	查看密码
		Mysql -u root -p 回车	
	找默认密码
		见mysql server安装	
		A RANDOM PASSWORD
	登录以后
	密码设置：
	改密码：set password=password('root')

	登录
	加入系统服务
		chkconfig --add mysql
	开机自启动 
		chkconfig mysql on
	开启远程访问
		
	登录
		mysql -uroot Cproot
	若：
	远程访问权限不足
	则：
	设置远程访问密码
		grant all privileges on *.* to 'root' @'%' identified by 'root';
	刷新
		flush privileges;
	设置防火墙(3306)
	添加到防火墙列表
		/sbin/iptables -I INPUT -p tcp --dport 3306 -j ACCEPT
	保存文件
		/etc/rc.d/init.d/iptables save 
	查看


	两种生效方法：
	    	如果需要关闭或开户防火墙：不用设置端口
			1) 重启后生效 
				开启： chkconfig iptables on 
				关闭： chkconfig iptables off 

			2) 即时生效，重启后失效 
				开启： service iptables start 
				关闭： service iptables stop
