# 首先新建数据库

找一个能用的数据库用来存储信息

新建一个数据库叫做gogs

# 运行docker容器

首先创建数据挂载目录,然后授予权限

```shell
mkdir -p /docker/gogs/data
chmod 777 /docker/gogs/data

然后我们直接运行容器
docker run -d -p 10022:22 -p 3000:3000 \
--name=gogs \
--privileged=true \
-v /docker/gogs/data/:/data/ \
gogs/gogs
```

# 测试完一键删除环境

删除docker

```shell
docker stop gogs
docker rm gogs
```

删除本地环境

```shell
rm -rf /docker/gogs/data
```

数据库需要自己删除
